Julia Gilstein's Editorial Website

View on Github View in Browser

This is a simple website that was used to promote Julia Gilstein's editing, copyediting, proofreading, and copywriting services for businesses and individuals. This was my second website that I designed for Julia Gilstein, but the first I coded.

About the Project

When I met with Julia, she did not need a content heavy, large, or complicated site. She needed something basic to showcase her services. I designed the site to be a single page that you could use navigation to reach each section easily or just scroll through. Designing was fun. I asked her what period in history she liked and she told me the Georgian time period. I designed her site with Georgian patterns and colours while keeping the design modern.

The Services section includes a long list of services. Instead of making a long text list, I wanted to make it visually appealing so I created, what I like to think of as, typewriter keys. Each circle contains an abbreviation of the service offered. Hovering over the service changes the "key" colour and clicking on any service "key" opens up an email with the subject line autocompleted to the specific service.

Her Editorial and Writing sections include examples of her work. Each example has a different colour. If more examples are added, the colours get repeated in a pattern of 4. Each example also includes a letter intital to break up the colour pattern and make it more visually appealing.

The site was designed with a mobile first approach. As the site expands, the logo changes from a horizontal logo to a stacked logo using JavaScript. A sticky navigation also appears when the user scrolls up to help them navigate faster.

What I Learned

When I first coded this site it was back in 2015. I had used images for all the initials, and made all of them background images that I could add in using CSS. There were also floats "floating" around, and it was not very good at being responsive. I have learned a lot since then!

While this site is not currently up anymore, I decided to go back and update it to reflect my HTML, CSS, and JavaScript knowledge. I used SASS to organize all the CSS, removed all floats and used flex-box to align content, and removed all the letter initial images replacing them with text. This made Julia Gilstein's site responsive and cleaner.
